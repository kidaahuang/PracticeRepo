# repo

## 網頁內容介紹與詳細 Review

### 1. 3D 客廳模擬圖（`3d-room.html`）
這個網頁以 Three.js 建構一個可互動的 3D 客廳場景，主體目標是把手繪平面示意轉為空間視覺化。場景設定包含霧化背景與柔和的環境光，並以 OrbitControls 提供旋轉、縮放、平移等操作，讓使用者能從不同角度檢視整體佈局。畫面中有木地板、左右牆面與開放式前方視角，並加入落地窗與窗框、玻璃材質，以及窗外的簡單天空背景，提升室內採光與空間感的呈現。

細節上，家具配置包含左側沙發、兩張圓形矮桌、右側電視櫃與電視、角落落地燈等，材質顏色與粗糙度都進行了對應設定，使木質、布料、玻璃與黑色塑料具有基本區分。右側牆面則特別建立可包含門洞的分段牆體，並用群組與轉軸設計出門片開啟的角度，反映實際室內空間。整體程式碼以多個建立函式拆分，結構清晰、易於後續擴充（例如加入更多家具或燈光效果）。

### 2. 3D Particle System（`particles.html`）
此頁面使用 OGL（輕量 WebGL 框架）打造粒子系統，呈現可旋轉、可移動視角的 3D 粒子場景。程式以 GLSL shader 定義粒子位置的波動邏輯，透過時間變化讓粒子做出呼吸與波浪式漂浮效果，同時依距離調整點大小，強化空間深度感。粒子數量預設為 2000，並以隨機色彩生成彩色點狀雲，呈現活潑而動態的視覺。

互動上支援滑鼠拖曳旋轉粒子群，並使用鍵盤方向鍵模擬第一人稱前進與轉向；UI 層提供按鍵提示，並且在 OGL 或 WebGL 載入失敗時顯示錯誤提示，提升可用性。整體結構將初始化、粒子建立、動畫迴圈、事件監聽與錯誤處理分離，程式碼條理清楚，適合再擴展為更多互動控制或不同粒子效果。

### 3. Snowy Forest Screensaver（`snowy-forest.html`）
這個網頁是一個完整的 2D Canvas 聖誕風雪森林螢幕保護程式。畫面包含夜空漸層、星空、月亮、遠景山丘、森林樹影、聖誕燈飾與雪地積雪效果，並帶有風向與重力影響的雪花飄落。程式設計了積雪高度地形（height map），會隨時間逐漸堆積、擴散與雪崩平衡，讓雪地有持續增厚的動態感。整體色彩與光影設計營造出冬夜節慶氣氛。

互動功能包含點擊切換提示顯示、鍵盤 L 開關燈飾、S 開關雪橇飛行、R 重置積雪、Esc 停止動畫。雪橇飛行與火花尾跡效果增加了節日的趣味性，也提升場景敘事感。程式碼採取模組化函式（如更新風、建構森林、繪製雪地等），並有大量可調參數（CONFIG），讓使用者易於微調視覺效果或效能表現，屬於內容豐富且完整的視覺展示頁面。
